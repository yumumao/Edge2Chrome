# Edge2Chrome

在Edge浏览器中检测指定链接，点击按钮用Chrome打开。支持通配符规则，可自定义匹配网址。

## 功能特性

- 🌐 **智能链接检测**: 自动检测页面中匹配规则的链接
- ⚙️ **灵活配置**: 支持通配符规则，可添加任意网站
- 🎯 **精确匹配**: 使用正则表达式进行精确的URL匹配
- 📱 **友好界面**: 直观的设置页面和弹出窗口
- 🔧 **高度自定义**: 可配置按钮文本、Chrome参数等
- 📋 **导入导出**: 支持配置的备份和分享
